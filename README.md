Инструкция по открытию и запуску кода
1. Требования
Для запуска программы необходимо:

Установленный Python 3.7 или новее
2. Сохранение кода
Скопируйте весь код из задания.
Откройте любой текстовый редактор.
Вставьте скопированный код.
Сохраните файл с расширением .py(Например Practice.py)
Откройте файл Practice.py в вашей среде разработки (например: Pycharm,  VS Code).
Нажмите зелёную кнопку запуска (▶️) или используйте сочетание клавиш (обычно Ctrl+F5).
